<dataFacet>

<dataFacet> Restricts the value of the strings used to represent values of a datatype, according to XML Schemas: Part 2: Datatypes. [22.5.3.1 Datatypes]
モジュールtagdocs — Documentation Elements
属性att.global (@xml:id, @n, @xml:lang, @xml:base, @xml:space) (att.global.rendition (@rend, @style, @rendition)) (att.global.linking (@corresp, @synch, @sameAs, @copyOf, @next, @prev, @exclude, @select)) (att.global.analytic (@ana)) (att.global.facs (@facs)) (att.global.change (@change)) (att.global.responsibility (@cert, @resp)) (att.global.source (@source))
name name¶the name of the facet.
状態 必須
データ型 teidata.word
正当な値:
length
minLength
maxLength
pattern
enumeration
whiteSpace
maxInclusive
minInclusive
maxExclusive
minExclusive
totalDigits
fractionDigits
value value¶the facet value.
状態 必須
データ型 string
上位
tagdocs: dataRef
下位空要素
解説

This element is only allowed when the parent dataRef refers with name to a datatype from the specification XML Schemas: Part 2: Datatypes.

例
<datatype>
 <dataRef name="decimal">
  <dataFacet name="maxInclusive"
   value="360.0"/>
  <dataFacet name="minInclusive"
   value="-360.0"/>
 </dataRef>
</datatype>
