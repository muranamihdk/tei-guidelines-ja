<refsDecl>

<refsDecl> (references declaration) 標準的な参照の作られ方を示す。 [2.3.6.3 Milestone Method 2.3 The Encoding Description 2.3.6 The Reference System Declaration]
モジュールheader — The TEI Header
属性att.global (@xml:id, @n, @xml:lang, @xml:base, @xml:space) (att.global.rendition (@rend, @style, @rendition)) (att.global.linking (@corresp, @synch, @sameAs, @copyOf, @next, @prev, @exclude, @select)) (att.global.analytic (@ana)) (att.global.facs (@facs)) (att.global.change (@change)) (att.global.responsibility (@cert, @resp)) (att.global.source (@source)) att.declarable (@default)
クラブのメンバー
model.encodingDescPart
上位
header: encodingDesc
下位
core: p
header: cRefPattern citeStructure refState
linking: ab
例
<refsDecl>
 <cRefPattern matchPattern="([A-Za-z0-9]+) ([0-9]+):([0-9]+)"
  replacementPattern="#xpath(//body/div[@n='$1']/div[$2]/div3[$3])"/>
</refsDecl>
bibliography 
一括表示 

This example is a formal representation for the referencing scheme described informally in the following example.

例
<refsDecl>
 <p>References are made up by concatenating the value for the
 <att>n</att> attribute on the highest level <gi>div</gi>
   element, followed by a space, followed by the sequential
   number of the next level <gi>div</gi> followed by a colon
   followed by the sequential number of the next (and lowest)
   level <gi>div</gi>.</p>
</refsDecl>
